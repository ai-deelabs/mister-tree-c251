This is a plain HTML/CSS/JS project. Build a landing page based on the
requirements below.

Requirements:
Mr. Tree (มิสเตอร์ทรี) is a health and wellness product brand targeting adults 40+ who value simplicity and clarity. The brand emphasizes clean, modern design with a light and airy aesthetic that feels contemporary and easy to navigate. The founder wants a minimalist approach with bright, soft tones that conveys trustworthiness and accessibility — no clutter, straightforward product presentation, and clear messaging for this mature demographic.

Output to a single file named `v3.html` with all CSS and JS
inline (no external files, no frameworks).

This is variant 3 of 3. Make it distinctly
different from the other variants in layout, palette, typography, and
visual direction.

No emojis anywhere. Use Lucide Icons instead:
`<script src="https://unpkg.com/lucide@latest"></script>` in head,
`<i data-lucide="icon-name"></i>` in body, `lucide.createIcons()` at end.

For images, use `https://picsum.photos/seed/{descriptive-keyword}/{width}/{height}`
(pick keywords that match the brand) or specific Unsplash URLs like
`https://images.unsplash.com/photo-{id}` if you know real photo IDs.

Constraints: no X-Frame-Options or frame-blocking headers (this page
will be loaded inside an iframe for client review).
